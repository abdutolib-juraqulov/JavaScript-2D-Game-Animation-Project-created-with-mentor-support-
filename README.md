# JavaScript-2D-Spielanimation-mit-Unterst-tzung-eines-Mentors-entwickelt-
2D-Browsergame mit HTML, CSS und JavaScript. Enthält Charakteranimationen, Gegnerbewegung, Kollisionserkennung, Soundeffekte und Game-Over-Logik. Entwickelt im Rahmen eines Lernprojekts mit Unterstützung eines Mentors zur Vertiefung meiner JavaScript- und Animationskenntnisse.
