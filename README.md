Orc Game Animation – JavaScript-Projekt

Dieses Projekt ist ein kleines 2D-Browser-Spiel, das mit HTML, CSS und JavaScript entwickelt wurde.
Es enthält Charakteranimationen, Gegnerbewegungen, Kollisionserkennung, Soundeffekte sowie ein einfaches Game-Over-System.

Funktionen des Spiels

Bewegung der Spielfigur mit den Pfeiltasten

Angriff mit der Taste „D“

Animierte Sprite-Grafiken

Kollisionserkennung mit getBoundingClientRect()

Zufälliges Spawnen des Gegners

Hintergrundmusik und Soundeffekte

Zustandsverwaltung (Angriff, Tod, Respawn, Game Over)

Technische Schwerpunkte

Animation über object-position

Nutzung von setInterval() für Spiel-Updates

Event-Handling über keydown und keyup

Logik für Zustände und Spielablauf
